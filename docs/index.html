C4/MIL Reader and Converter to PDF

Select one legacy C4/JEDMICS drawing or MIL file containing C4/CCITT4 data, preview the generated PDF, then download it. Or select a folder to batch convert all C4/MIL files found in that folder and its subfolders. Files are processed locally in your browser and are not uploaded.

Drop one .C4 or .MIL file here or use the file picker

Folder mode scans subfolders for .C4 and .MIL files, converts them locally, and downloads one ZIP containing the generated PDFs plus a conversion report. Browsers do not allow this page to write PDFs directly back into your original folders. 



      

        
What this page version does

        

          The browser version can convert one file for preview/download, or batch convert a
          selected folder tree into one ZIP download. The desktop Python release remains the
          full tool for writing converted PDFs beside source files in local folders.
        

        

          The generated PDF embeds the original CCITT Group 4 tiles directly, so the page
          does not need to upload files or run a server-side converter.
